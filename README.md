# Wipro Training Projects

This repository contains assignments and practice projects completed during the Wipro training program.

## Projects Included

### 1. Assignments

Practice assignments completed during training covering core .NET and programming concepts.

### 2. Unit Testing Practice

Projects created while learning unit testing in .NET using:

* xUnit
* Moq
* ASP.NET Core testing practices

### 3. DriveEase – Car Rental Management System

DriveEase is a car rental system developed using ASP.NET Core MVC and Entity Framework.

**Features**

* Customer Management
* Lease Management
* Vehicle Management
* Payment Processing
* Revenue Reporting

**Technologies Used**

* C#
* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* xUnit
* Moq
